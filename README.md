# 🎯 MÖLKKY CRICKET - SCOREBOARD PRO

モルックの派生競技として非常に戦略的で人気のある**「モルックリケット（Mölkky Cricket）」**に完全特化した、ブラウザで動く本格的なデジタルスコアボードアプリです。

ユニバーサルデザインを意識した直感的な**「融合型メインボード UI」**を採用し、屋外のタフな試合環境でも1タップで正確かつスピーディーにスコアを記録・管理できます。

---

## ✨ 主な機能

* **2つの対戦モード対応**
    * **1試合対戦（クイックマッチ）:** 1ゲーム（12ターン）完結の手軽なモード。
    * **セット対戦（セットマッチ）:** 2本先取（BO3）や3本先取（BO5）に対応。セット間の先攻・後攻選択や、トータル履歴の蓄積が可能。
* **革新的な「融合型メインボード UI」**
    * 盤面のスキットル番号自体をタップして「的中ターゲット」を選択する直感的な操作感。
    * 1つの画面で「現在のマーク状況」と「入力インターフェース」をシームレスに統合。
* **公式＆カスタムルールに完全準拠**
    * **先攻1ターン目の特別ルール**（マークなし、倒した本数がそのままダイレクトに得点）の自動適用。
    * **複数本的中（MULTI HIT）時の正確な計算**（自分がオープンしているスキットルは1本＝1点として本数加算、未オープンは各1マーク付与）。
    * 数字ごとの「オープンに必要なマーク数（2点/3点）」の個別・一括カスタマイズ機能。
    * DOUBLE (✕2) / TRIPLE (✕3) 的中アクションパネル。
* **優れた視認性と演出**
    * 屋外の直射日光下でも見えやすい**ユニバーサル記号表示**（/、✕、⨂）を採用。
    * 手番プレイヤーの超絶強調表示（ネオンエフェクト）およびダークモード切り替え。
    * ターン終了時に投擲結果がふわっと浮かび上がる**「符号演出アニメーション（HOWAN）」**。
* **安心のリカバリー機能**
    * 押し間違えても瞬時に一手前に戻れる高精度な **UNDO（1手戻る）** 機能。
    * 試合中のQUIT（離脱）やRESET（ゲームやり直し）の誤動作を防ぐ確認ダイアログ。
* **本格的なスタッツ・履歴分析**
    * 各ターン・各投の的中マークと獲得点数、累計スコアをグリッド表示する **SCORE HISTORY テーブル**。
    * 1ターンあたりの平均マーク数や、最大連続ミス（MISS）数など、実力分析に役立つ試合スタッツ算出。

## 🛠 使い方

1.  **トップ画面:** 「1試合対戦」か「セット対戦」を選択します。
2.  **設定画面:** * プレイヤー名を入力します（🎲ボタンで先攻・後攻のシャッフルが可能）。
    * マッチ形式、ボーナスマーク（DOUBLE/TRIPLE）の有無、1ターンあたりの投擲数（2本/3本）、スキットルのオープン条件を設定し、`GAME START` をタップ。
3.  **試合画面:**
    * 倒したスキットルを盤面からタップして選択します。
    * **1本のみ的中（SINGLE）の場合:** 番号を1つ選択 ➔ 下部の `SINGLE/DOUBLE/TRIPLE` を選択 ➔ `🎯 SINGLE確定` をタップ。
    * **複数本的中（MULTI）の場合:** 倒したすべての番号を選択 ➔ `👥 MULTI確定` をタップ。
    * **ミスの場合:** ボードは選択せず、一番下の `💀 MISS / 0 MARKS` をタップ。

## 📐 モルックリケット 基本ルール（アプリ搭載仕様）

1.  **勝利条件:**
    * 12ターン終了時点で相手より高得点であること（同点の場合はオープン状態のキープ数が多い方が勝ち）。
    * **【コールド勝ち】** 相手のスコアと同点以上の状態で、自分の全てのスキットル（6〜12番）をオープンにした瞬間に勝利。
2.  **オープンとクローズ:**
    * フラット状態（⚪️）から既定マークに達すると**オープン（🟢）**となり、自分の陣地（加点対象）になります。
    * 相手がオープンしているスキットルに自分も既定マークを重ねると**クローズ（🔴）**となり、そのスキットルは盤面から除外（両者加点不可）されます。
