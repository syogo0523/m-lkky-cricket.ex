# 🎯 モルックリケット (Mölkky Cricket) - スコアボード

モルックの投擲技術と、ダーツの「クリケット」の戦略性を融合させたアドバンスド・ボードゲーム『モルックリケット』専用のデジタルスコアボードアプリです。
通常のルールに加え、投擲回数や目標マーク数のカスタマイズ、各種ボーナスルールのON/OFFなど、プレイヤーのレベルに合わせた柔軟なゲーム設定を搭載したパワーアップバージョンです。

## 🌟 主な機能と特徴

- **豊富なカスタムオプション**: ゲーム開始前に、1ターンの投擲回数（2投/3投）、各スキットルの必要マーク数（2つ/3つ）、ボーナスの有無を自由に設定可能。
- **洗練されたスポーツテックUI**: プレイヤーA（赤）とプレイヤーB（青）のカラーリング、およびスキットル状況（フラット、1マーク、2マーク、オープン、クローズ）がひと目でわかるスマートな配色。
- **厳密なクリケットルール計算**: 
  - 自分がオープンした段階で、相手が未オープンの場合の超過マークは「すべて得点」に加算。
  - 自分がオープンした瞬間に、相手もすでにオープンしていた場合は即座に「CLOSED」となり、超過マークは「自動的に無効（切り捨て）」となる公式準拠のロジック。
- **誤タップ救済（Undo機能）**: 試合中だけでなく、勝利画面（結果モーダル）からでも「ひとつ前に戻る」ことができ、誤入力によるゲーム終了を防ぎます。
- **高度な試合スタッツ（分析機能）**: 
  - 1ターン（1プレイ）あたりの平均マーク数を自動算出。
  - いずれかのプレイヤーが「6つのスキットル」をオープンした瞬間にスタッツ計算がロック（フリーズ）され、中盤までの純粋な投擲精度を正確に評価します。
- **完全レスポンシブ & ダークモード対応**: スマホのブラウザで最も見やすくなるよう設計。夜間や屋外のプレイにも最適なダークモードを搭載。

## 📋 ゲームルール概要

### 勝利条件
1. **12ターン終了時**に、相手より高いスコアを獲得していること（同点の場合はオープンしているスキットル数が多い方の勝利）。
2. 相手よりスコアが高い状態で、**すべての対象スキットルを「OPEN」**にすること（コールド勝ち）。

### 基本用語
- **OPEN（オープン）**: 対象のスキットルに対して、設定された必要マーク数（2または3）に達すると、そのスキットルは自分のもの（オープン状態）になり、以降その数字を倒すたびに得点が入ります。
- **CLOSED（クローズ）**: 両プレイヤーが同じスキットルをオープンした場合、そのスキットルは「クローズ（封印）」され、以降はどちらのプレイヤーも得点できなくなります。
- **BONUS (×2 / ×3)**: 単品的中時に、条件（直当て等）を達成した場合に2マークまたは3マーク分を一度に獲得できるシステムです。

### 特殊ルール
- **先攻1ターン目の制限**: 先攻の第1ターン目のみ、スキットルへのマークは行われず、倒した「本数」がそのままスコア（得点）になります（後攻の1ターン目から通常ルールが適用されます）。
